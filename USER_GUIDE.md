# 식당 QR 오더 - 사용자 설명서

## 📱 앱 소개

이 앱은 **QR 코드로 테이블에서 직접 주문하는 시스템**입니다.
- **손님**: 스마트폰으로 QR 코드를 스캔 → 메뉴 선택 → 주문
- **업주**: 관리자 페이지에서 주문 수신, 조리 상태 관리

---

## 👥 사용자별 가이드

### 1️⃣ 손님 (고객) 사용법

#### 시작하기
1. 테이블의 **QR 코드를 스마트폰 카메라로 스캔**
2. 자동으로 주문 앱 화면 열림
3. 테이블 번호가 자동으로 설정됨

#### 주문하기
1. **메뉴 카테고리 선택** (탕수육, 요리, 면, 밥, 세트, 추가)
2. 원하는 메뉴 찾기
3. **"담기" 버튼 클릭**
4. 수량 확인 후 다시 클릭하면 추가됨

#### 주문 확인 & 변경
1. 화면 우측 하단 **장바구니 아이콘** 클릭
2. 선택한 메뉴 목록 확인
3. **수량 조절**: +/- 버튼으로 증감
4. **삭제하기**: 항목 옆의 X 버튼 클릭
5. **총 가격** 확인

#### 주문 완료
1. 장바구니에서 **"주문하기" 버튼** 클릭
2. **"주문이 완료되었습니다!"** 메시지 확인
3. 업주가 주방에서 받은 주문 조리 시작
4. 조리 완료 시 호출 또는 진동 알림

#### 팁
- 페이지를 새로고침해도 주문 내역이 **자동 저장**됨
- 같은 메뉴를 여러 번 "담기"하면 수량 자동 증가
- 주문 후 메뉴 추가 가능

---

### 2️⃣ 업주 (관리자) 사용법

#### 관리 페이지 열기
1. 포스기 또는 업무용 컴퓨터에서 **admin.html** 열기
2. 관리자 대시보드 화면 표시됨

#### 주문 수신
- 손님이 주문하면 **자동으로 알림 표시**
- 알림음 🔔이 울림 (필요시 음소거 가능)

#### 주문 상태 관리

| 상태 | 의미 | 버튼 |
|------|------|------|
| **대기중** | 새 주문 수신 | "접수" 클릭 |
| **조리중** | 주방에서 조리 중 | "완료" 클릭 |
| **완료** | 조리 끝남, 호출 대기 | - |

#### 주문 필터링
1. 상단 탭에서 **"전체 / 대기중 / 조리중 / 완료"** 선택
2. 해당 상태의 주문만 표시됨

#### 실시간 통계
대시보드에서 확인:
- **총 주문 수**
- **조리중 건수**
- **완료 건수**
- **오늘의 매출**

#### 완료 주문 정리
1. **"완료 주문 삭제"** 버튼 클릭
2. 완료된 모든 주문 일괄 삭제

#### 알림 설정
- 우측 하단 🔔 버튼으로 알림음 **ON/OFF**
- 브라우저 알림 허용 권유

## 🎮 기능별 빠른 참조

### 메뉴 카테고리

| 카테고리 | 예시 메뉴 | 가격대 |
|---------|---------|------|
| **탕수육** | 소/중/대/특 | 15,000~30,000원 |
| **요리** | 양장피, 팔보채, 깐풍육 등 | 24,000~50,000원 |
| **면** | 짜장면, 짬뽕, 우동 등 | 7,000~13,000원 |
| **밥** | 볶음밥, 짬뽕밥, 잡채밥 등 | 8,000~11,000원 |
| **세트메뉴** | 짬뽕+탕수육 조합 | 17,000~28,000원 |
| **추가** | 술, 음료 | 2,000~6,000원 |

## ❓ 자주 묻는 질문

### Q1. QR 코드를 잃어버렸어요
**A.** qr-generator.html을 열어 다시 생성하세요. 테이블 번호와 URL을 입력하면 됩니다.

### Q2. 주문했는데 업주가 못 받았어요
**A.** 
- 손님: "주문하기" 버튼을 정확히 눌렀는지 확인
- 업주: admin.html을 열었는지 확인, 새로고침 시도

### Q3. 같은 메뉴를 여러 개 주문하려면?
**A.** 메뉴 "담기" 버튼을 여러 번 클릭하면 수량이 자동 증가합니다.

### Q4. 주문 후 취소할 수 있나요?
**A.** 주문 완료 후에는 업주에게 직접 말씀해주세요.

### Q5. 노트북이나 태블릿으로도 되나요?
**A.** 네, 모든 스마트 기기의 웹 브라우저에서 작동합니다.

### Q6. 업주 페이지의 알림음이 안 들려요
**A.** 
- 브라우저 볼륨 확인
- 🔔 버튼이 켜져 있는지 확인
- 브라우저 음소거 해제

## 🔧 트러블슈팅

### 문제: 주문 목록이 안 보여요
**해결책:**
1. 브라우저 새로고침 (F5)
2. 캐시 삭제: Ctrl+Shift+Del
3. 다른 브라우저 시도

### 문제: 업주가 주문을 못 받아요
**해결책:**
1. admin.html 페이지 새로고침
2. 같은 WiFi 네트워크에 연결되어 있는지 확인
3. 브라우저 개발자 도구(F12)에서 오류 확인

### 문제: QR 코드가 인식이 안 돼요
**해결책:**
1. 카메라 렌즈 깨끗이 닦기
2. 밝은 환경에서 시도
3. 수동으로 URL 입력하기

## 📊 운영 팁

### 테스트 방법
1. **두 브라우저 창 열기**
   - 창1: index.html (손님 역할)
   - 창2: admin.html (업주 역할)
2. 손님이 주문 → 업주 창에서 실시간 수신 확인

### 일일 체크리스트
- [ ] admin.html 페이지 열기
- [ ] 알림음 ON 확인
- [ ] 새 주문 테스트
- [ ] 일일 종료 시 "완료 주문 삭제" 실행

### 메뉴 변경하기
메뉴를 추가하거나 가격을 바꾸려면:
1. index.html을 텍스트 편집기로 열기
2. `<div class="menu-item">` 부분 찾아서 수정
3. 파일 저장 후 브라우저 새로고침
