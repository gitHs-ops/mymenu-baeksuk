시스템 개요

🎯 목적

백숙 전문점을 위한 비대면 테이블 주문 시스템으로, QR 코드를 통해 손님이 직접 주문하고 주방에서 실시간으로 주문을 받을 수 있는 스마트 오더 솔루션입니다.

💡 핵심 가치

  • ✅ 인건비 절감: 주문 받는 직원 최소화
  • ✅ 주문 정확도 향상: 손님이 직접 입력하여 오더 실수 방지
  • ✅ 회전율 증가: 빠른 주문 처리로 테이블 회전율 향상
  • ✅ 비대면 서비스: 코로나 이후 선호되는 비접촉 주문 방식
  • ✅ 실시간 관리: 주문 현황을 한눈에 파악

주요 기능

👥 고객용 주문 화면

1. 간편한 주문 과정

테이블 QR 스캔 → 메뉴 선택 → 장바구니 담기 → 주문 완료

2. 주요 특징

  • 📱 스마트폰 전용: 손님 핸드폰에서 편리하게 사용
  • 🛒 장바구니 기능: 여러 메뉴를 한번에 주문
  • ➕➖ 수량 조절: 간편한 수량 증감
  • 💰 자동 금액 계산: 주문 전 총액 자동으로 확인
  • 🔄 주문 내역 보관: 실수로 화면 꺼져도 주문 내역 유지
  • 🎨 쉬운 화면 구성: 누구나 쉽게 사용 가능

▲ 고객 스마트폰 화면 예시

3. 메뉴 카테고리

  • 🍲 메인메뉴 (토종닭 백숙, 오리 백숙, 토종옻닭 백숙, 오리불고기 등)
  • ⭐ 특선 (장수탕 닭 또는 오리)
  • ➕ 추가메뉴 (왕새우, 왕홍합, 왕우무, 전복, 가리비, 죽 포장)
  • 🥗 사이드 (도토리묵무침)
  • 🍺 주류 (소주, 맥주, 막걸리, 음료수)

👨‍💼 업주용 관리 화면

▲ 업주용 주문 관리 화면 예시

1. 주문 받기 (실시간)

  • 🔔 즉시 알림: 새 주문 들어오면 소리 + 화면 알림
  • 📊 주문 현황판: 한눈에 보는 주문 상태
  • 🎯 버튼 한번으로 상태 변경: 접수 → 조리중 → 완료
  • 🔍 주문 분류 보기: 상태별로 주문 확인 (대기중/조리중/완료)
  • 📋 주문 자세히 보기: 테이블 번호, 메뉴, 수량, 금액

2. 매출 및 주문 현황

  • 💵 오늘 매출: 오늘의 총 매출 확인
  • 📈 주문 개수: 총 주문 수, 상태별 주문 수
  • 🕐 시간대별 확인: 손님 많은 시간대 파악
  • 📊 인기 메뉴: 많이 팔린 메뉴 확인

3. 편의 기능

  • 🔕 알림음 ON/OFF: 상황에 맞게 조절
  • 🗑️ 완료 주문 정리: 일괄 삭제 기능
  • 💾 자동 저장: 주문 내역 자동으로 저장
  • 🖥️ 여러 기기 사용 가능: 컴퓨터, 태블릿 모두 사용 가능

기대 효과

💰 비용 절감

1. 인건비 절감

  • 기존: 홀 서빙 직원 2명 필요
  • 도입 후: 1명으로 운영 가능
  • 절감액: 월 200만원 이상

2. 주문 실수 감소

  • 기존: 잘못 들은 주문으로 재조리 발생
  • 도입 후: 손님이 직접 입력하여 실수 최소화
  • 절감액: 월 50만원 이상 (재료비 절감)

📈 매출 증대

1. 회전율 향상

  • 기존: 주문 대기 시간 5-10분
  • 도입 후: 즉시 주문 가능
  • 효과: 테이블 회전율 20% 향상

2. 추가 주문 증가

  • 기존: 직원 부르기 부담스러워 추가 주문 망설임
  • 도입 후: 언제든 편하게 추가 주문
  • 효과: 추가 주문 30% 증가

😊 고객 만족도 향상

1. 편리한 주문

  • 직원 부르지 않고 편하게 주문
  • 메뉴를 천천히 보고 선택 가능
  • 가격을 미리 확인하고 주문

2. 빠른 서비스

  • 주문 즉시 주방 전달
  • 대기 시간 단축
  • 정확한 주문 처리

📊 매출 관리 편리

1. 매출 확인

  • 오늘의 매출을 바로바로 확인
  • 시간대별 매출 확인
  • 목표 대비 얼마나 팔렸는지 확인

2. 메뉴 분석

  • 어떤 메뉴가 인기 있는지 확인
  • 안 팔리는 메뉴 개선
  • 메뉴판 구성 개선

자주 묻는 질문

Q1. 기존 포스기와 연결이 되나요?

A. 현재는 따로 작동하며, 포스기 연결은 추가 작업이 필요합니다. 대부분의 포스기(포스피, 오케이포스 등)와 연결 가능하며, 별도 비용이 듭니다.

Q2. 인터넷이 끊기면 어떻게 되나요?

A. 주문 내역은 임시로 저장되며, 인터넷 다시 연결되면 자동으로 전송됩니다. 안정적인 서비스를 위해 백업 인터넷(핸드폰 핫스팟 등)을 권장합니다.

Q3. 손님이 핸드폰이 없으면 어떻게 하나요?

A. 기존 방식(직접 주문)과 같이 운영 가능합니다. QR 주문과 직접 주문을 모두 관리 화면에서 확인할 수 있습니다.

Q4. 메뉴 가격이나 항목을 바꿀 수 있나요?

A. 네, 관리 화면에서 언제든지 메뉴를 추가/수정/삭제할 수 있습니다. 바로 반영됩니다.

Q5. 결제는 어떻게 하나요?

A. 현재는 주문만 받고, 결제는 기존 방식(카운터 결제)을 유지합니다.

Q6. 다른 매장에서도 쓸 수 있나요?

A. 네, 메뉴만 바꾸면 어떤 업종(한식, 일식, 카페 등)에서도 사용 가능합니다.

Q7. 주문 내역은 얼마나 보관되나요?

A. 모든 주문 내역은 계속 보관되며, 언제든지 확인 가능합니다. 매출 확인 및 세금 자료로 쓸 수 있습니다.

Q8. 여러 직원이 동시에 관리 화면을 쓸 수 있나요?

A. 네, 여러 기기에서 동시 접속 가능하며, 바로바로 업데이트됩니다.

Q9. 배달 주문도 받을 수 있나요?

A. 현재는 테이블 주문 전용이지만, 배달 주문 기능 추가 개발 가능합니다.

Q10. 시스템 사용이 어렵지 않나요?

A. 매우 쉽게 만들어져 있어, 핸드폰 쓸 줄 아시면 누구나 쉽게 사용할 수 있습니다. 설치할 때 충분히 교육해드립니다.
